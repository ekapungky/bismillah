Peta Interaktif Fasilitas Kesehatan

Interface dirapikan agar peta menjadi fokus utama. Filter, mode data, tampilan peta, informasi penggunaan, dan legenda dikelompokkan lebih ringkas tanpa mengubah data dan fungsi utama yang sudah ada.

Mode Data

Pilih tampilan data yang ingin digunakan pada peta.

Filter ini berlaku untuk data fasilitas kesehatan Surakarta.

Pilih basemap sesuai kebutuhan visualisasi dan kenyamanan tampilan.

Klik marker atau area layer untuk melihat detail data. Fitur rute berlaku untuk marker fasilitas Surakarta.

  • Pilih mode data yang ingin ditampilkan
  • Gunakan filter fasilitas Surakarta
  • Cari nama atau alamat fasilitas
  • Ubah tampilan peta sesuai kebutuhan
  • Tentukan rute ke lokasi tujuan
