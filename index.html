Tentang WebGIS Ini

Halaman beranda difokuskan untuk pengantar, sedangkan seluruh kontrol peta dipindahkan ke halaman peta agar tampilan tidak menumpuk. Dengan begitu, halaman utama menjadi lebih bersih dan halaman peta bisa difokuskan khusus untuk eksplorasi data spasial.

Halaman Beranda

Digunakan untuk pengantar, deskripsi singkat, dan ringkasan informasi utama sebelum pengguna masuk ke halaman peta.

Halaman Peta

Berisi seluruh kontrol utama seperti mode data, filter, pencarian, basemap, legenda, serta visualisasi peta interaktif.

Data Tetap Sama

Perubahan hanya pada interface dan pengalaman penggunaan. Struktur data GeoJSON dan data QGIS2Web tetap dipakai seperti sebelumnya.
